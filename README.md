# Pharma Multi-Agent System

## Overview

Pharma Multi-Agent System is an Agentic AI application built using Streamlit, Groq LLM, LangChain, PubMed, and DuckDuckGo Search.

The application utilizes multiple specialized AI agents to perform pharmaceutical research, medical analysis, drug interaction assessment, regulatory review, and professional report generation.

## Features

* Planner Agent
* Research Agent
* Medical Agent
* Drug Interaction Agent
* Regulatory Agent
* Report Generation Agent
* PubMed Integration
* Web Search Integration
* PDF Report Export
* Streamlit Interface

## Tools used

* Python
* Streamlit
* LangChain
* Groq API
* PubMed
* DuckDuckGo Search
* ReportLab
